📂 File Explorer (CLI)

A terminal-based File Explorer built with Python.

Features

- Browse available drives.
- Navigate through folders.
- Display files and folders with sizes.
- Select a working directory.
- Go back to parent directories.
- Interactive terminal UI with colored output.
- Modular project structure.
